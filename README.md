# Hurry-up
Репозиторий команды КиберЛаб с игрой Поспеши
Поздний вечер. Ты подъезжаешь к старому заброшенному дому на отшибе — и в этот момент твоя машина глохнет.
Выбора нет. Нужно войти внутрь и найти инструменты, чтобы починить её и убраться отсюда как можно скорее.
Дом встречает тебя тишиной и темнотой. Ты обыскиваешь комнаты, шаришь по ящикам и полкам в поисках нужных вещей. Каждая секунда на счету.
Но чем дольше ты остаёшься — тем сильнее ощущение, что ты здесь не один.
Найди инструменты. Почини машину. Уезжай.
Пока не стало слишком поздно.
